## 產品快速選擇指引

當顧客說「我想買床墊」但沒有明確指向時，用以下問句引導：

1. 「你平常比較常仰睡、側睡、還是趴睡呢？」
2. 「睡覺的時候會不會容易覺得熱？」
3. 「有沒有腰痠背痛或肩頸不舒服的困擾？」
4. 「你喜歡比較有支撐感的，還是比較軟、有包覆感的？」

根據回答的配對邏輯：

| 顧客回答 | 優先推薦 |
|---------|---------|
| 仰睡 + 腰痠 + 喜歡硬一點 | 山丘樂眠床 |
| 怕熱 + 淺眠 + 腦袋停不下來 | 冰島樂眠床 |
| 側睡 + 肩頸壓力 + 喜歡軟包覆 | 飄雲樂眠床 |
| 預算有限 / 租屋 / 學生 | 無光薄墊 or 無光厚墊 |

**重要**：這是「建議方向」不是「標準答案」。永遠鼓勵顧客到門市試躺。
