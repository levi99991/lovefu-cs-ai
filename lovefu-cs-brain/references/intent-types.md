# 意圖類型定義｜intent-types.md

## 8 種意圖

### ORDER — 訂單查詢

顧客想知道訂單的狀態、出貨進度、物流追蹤。

判斷範例：
- 「我的東西寄了嗎」
- 「訂單到哪了」
- 「什麼時候會收到」
- 「出貨進度」
- 「追蹤碼是多少」
- 「#LF2026001」（直接丟訂單編號）

### PRODUCT — 產品諮詢

顧客詢問產品的規格、價格、差異比較。

判斷範例：
- 「山丘跟冰島差在哪」
- 「你們床墊多少錢」
- 「有什麼尺寸」
- 「月眠枕怎麼樣」
- 「無光薄墊跟厚墊差在哪」

### SLEEP — 睡眠諮詢

顧客有睡眠困擾，需要輔睡員引導和建議。

判斷範例：
- 「我睡覺腰很痠」
- 「晚上很容易熱醒」
- 「翻來翻去睡不好」
- 「肩膀很緊落枕」
- 「要選什麼床比較好」
- 「我該怎麼改善睡眠」

### RETURN — 退換貨

顧客要退貨、換貨、或詢問安心睡計畫規則。

判斷範例：
- 「我想退床墊」
- 「安心睡怎麼退」
- 「收到有瑕疵」
- 「不滿意想退貨」
- 「退換貨要準備什麼」

### COMPLAINT — 客訴

顧客抱怨、情緒激動、要找主管。

判斷範例：
- 「你們態度很差」
- 「等了一個月還沒收到太誇張」
- 「我要投訴」
- 「找你們主管」

注意：COMPLAINT 和 RETURN 有時候重疊。
如果顧客情緒平穩只是要退貨 → RETURN。
如果顧客情緒激動而且要退貨 → COMPLAINT（直接轉人工）。

### STORE — 門市查詢

顧客詢問體驗店的位置、營業時間、預約方式。

判斷範例：
- 「你們門市在哪」
- 「桃園店幾點開」
- 「怎麼預約試躺」
- 「台中有店嗎」

### MEMBER — 會員查詢

顧客詢問會員等級、點數、優惠。

判斷範例：
- 「我有多少點數」
- 「怎麼升等」
- 「會員折扣怎麼算」
- 「點數什麼時候到期」

### CHAT — 閒聊

打招呼、感謝、告別、與大島無直接關係的閒聊。

判斷範例：
- 「你好」
- 「謝謝」
- 「晚安」
- 「你是 AI 嗎」
- 「今天天氣好好」
